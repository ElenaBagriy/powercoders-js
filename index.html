1. Put the inputed numbers into array.
  2. Create an empty object with properties value, left and right.
  3. Loop through all the numbers that need to be sorted.
  4. For each number:
    1. Stick it into the object. If the object is empty, this will be the first object value.
    2. If the object is not empty, compare the number at each node(value):
    3. If the number is less than or equal to the current node, go to the left subtree.
    4. If the number is bigger than the current node, go to the right subtree.
    5. Continue this process until you find a suitable place for the number.
  5. Once all the numbers have been added to the tree, go through your entire tree starting at the top of it to get a sorted list of values. The algorithm:
    1. Create an empty array for the sorted list.
    2. If the node has the left-property, go there. If not - go to next step. If there are more "left-branches" - go down there. After you find the most left branch, that exists and was not visited before - push the value of this left-property-branch into an empty array.
    3. Go upper and push the value into the array.
    4. After checking the left-property and pushing the own-value, go to the right branch.If not go to next step
    5. After checking the right-property, repeat point 5.c.
    6. If you need to sort number in descending order, always start with the right branch instead of the left.
  6. Print a sorted list.
